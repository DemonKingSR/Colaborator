## 🎯 Animation Details

### Timing Functions
- `ease-out`: For entrances (0.6s - 0.8s)
- `ease-in-out`: For loops (1.5s - 2s)
- `cubic-bezier(0.175, 0.885, 0.32, 1.275)`: For bouncy effects

### Key Animations
1. **fadeIn**: Opacity 0 to 1 with translateY
2. **slideUp**: Bottom to top entrance
3. **slideDown**: Top to bottom entrance
4. **scaleIn**: Scale 0.8 to 1 with opacity
5. **pulse**: Continuous scale animation
6. **shimmer**: Loading skeleton effect
7. **gradientShift**: Animated gradient border
8. **shake**: Error state animation
